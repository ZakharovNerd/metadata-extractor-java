Загрузить документ из файла

Проблема

У вас есть файл на диске, содержащий HTML, который вы хотите загрузить и проанализировать, а затем, возможно, обработать или извлечь из него данные.

Решение

Используйте статический Jsoup.parse(File in, String charsetName, String baseUri)метод:

File input = new File("/tmp/input.html");
Document doc = Jsoup.parse(input, "UTF-8", "http://example.com/");

Описание

Метод parse(File in, String charsetName, String baseUri)загружает и анализирует файл HTML. Если при загрузке файла возникает ошибка, он выдает ошибку IOException, которую следует обработать соответствующим образом.

Параметр baseUriиспользуется синтаксическим анализатором для разрешения относительных URL-адресов в документе до того, как <base href>элемент будет найден. Если это вас не беспокоит, вместо этого вы можете передать пустую строку.

Существует родственный метод, parse(File in, String charsetName)который использует местоположение файла в качестве baseUri. Это полезно, если вы работаете с локальным сайтом файловой системы, и относительные ссылки, на которые он указывает, также находятся в файловой системе.
